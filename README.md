# Lurker

## Installation

```bash
pip install scrapy
pip install validators
```

## Usage

```bash
scrapy crawl beyond -o teams.csv -a url="http://teambeyond.net/forum/tournaments/standings/160-astronauts-2000-rocket-league-3v3-1126-700pm-est/"
```
